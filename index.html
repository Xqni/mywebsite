About me


I am curios person who wants to know so many things. I have tried couple of things in my 20-years of life such as video editing (anime video editor btw) - also did as a freelancing side hustle, two security jobs, a coorporate job (still doing it), two side businesses - you can find the link to one (my shop) on the top.

Why did I go out of my way to do all this you might ask?

I heard this from someone that you won't know if its for you until you try, and not just try but go all in.

Everything you are trying to do, someone has already done. There is a blueprint to the things you want to achieve. To find that blueprint, you have to be willing to step out. That is what means to grow!

Task Tracker

Python


I made this simple, beginner Python project. This program is a command line program which, as the name suggests, tracks user's tasks. Users can add, update, delete, list, and mark (done or in-progress) tasks using defined commads. I have also included a separate help text file which users can either go and read or can use the help command to print all the text in the file to their terminal. Make sure to check it out using the link below.
